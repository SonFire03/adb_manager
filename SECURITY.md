# Security Policy

## Scope

ADB Manager Pro is a local Android administration/debugging tool.
It does **not** target exploitation, bypass, unauthorized access, or root circumvention.

## Supported versions

The `main` branch is the only actively supported line at this stage.

## Reporting a vulnerability

Please open a private security report through GitHub Security Advisories when possible.
If unavailable, open an issue with minimal sensitive detail and request private follow-up.

## Responsible usage

- Use only on devices you own or are explicitly authorized to manage.
- Keep USB/Wi-Fi debugging enabled only when required.
- Revoke unknown ADB authorizations on devices periodically.
